# @weiui/react

## 0.0.1

- Initial pre-release: 65 styled React components built on `@weiui/headless` and `@weiui/css`. Includes overlays (Dialog, Drawer, Popover, Menu, Tooltip, Toast, CommandPalette), forms (Input, Field, Checkbox, Switch, RadioGroup, Select, Slider, Rating, ColorPicker, AutoComplete, MultiSelect, DatePicker, Calendar, InputNumber, InputOTP, FileUpload), data (DataTable, TreeView, Transfer, Timeline, Chart, Editor), display (Card, Badge, Avatar, Chip, Skeleton, EmptyState, ProgressBar, Spinner, Alert, Code, Kbd), navigation (Tabs, Breadcrumb, Pagination, Stepper, Sidebar, AppBar, BottomNav, Menu, Link, SpeedDial), layout (Container, Grid, Stack, Divider, Spacer, AspectRatio, Splitter, Accordion), typography (Heading, Text, Label, VisuallyHidden), and action surfaces (Button, ButtonGroup, ToggleGroup).
