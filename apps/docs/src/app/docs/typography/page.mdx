# Typography

WeiUI ships three type stacks exposed as CSS variables:

- `--wui-font-family-sans` — **Inter** (body, UI, prose)
- `--wui-font-family-display` — **Instrument Serif** (hero/display headings)
- `--wui-font-family-mono` — **JetBrains Mono** (code)

## Display typography

The hero on the landing page uses the display stack. Add `className="wui-display"` to any heading to opt in.

<div style={{ padding: "2rem", border: "1px solid var(--wui-color-border)", borderRadius: "var(--wui-shape-radius-lg)" }}>
  <h2 className="wui-display" style={{ fontSize: "var(--wui-font-size-5xl)", lineHeight: 1.05, letterSpacing: "-0.02em" }}>
    A design system that earns its place.
  </h2>
</div>

## Line height

| Token | Value | Description |
|-------|-------|-------------|
| `--wui-font-lineHeight-tight` | 1.25 | Headings and display text |
| `--wui-font-lineHeight-snug` | 1.375 | UI text, labels, buttons |
| `--wui-font-lineHeight-normal` | 1.5 | Body copy — comfortable reading |
| `--wui-font-lineHeight-relaxed` | 1.625 | Long-form prose |
| `--wui-font-lineHeight-loose` | 2 | Dense vertical rhythm (e.g. spaced list items) |

## Mono stack

Block + inline code use `--wui-font-family-mono`.

<pre style={{ fontFamily: "var(--wui-font-family-mono)", fontSize: "var(--wui-font-size-sm)", background: "var(--wui-color-muted)", padding: "var(--wui-spacing-3)", borderRadius: "var(--wui-shape-radius-md)", margin: 0 }}>
{`const greet = (name: string) => \`Hello, \${name}!\`;`}
</pre>
