## Props

| Component | Props | Description |
|-----------|-------|-------------|
| Card | HTMLDivElement attrs | Root container |
| CardHeader | HTMLDivElement attrs | Top section |
| CardContent | HTMLDivElement attrs | Main content area |
| CardFooter | HTMLDivElement attrs | Bottom section |
| Badge | variant, color | Status label |
| Avatar | size | User/entity avatar |
| Skeleton | variant | Loading placeholder |

## Accessibility

- Card uses `container-type: inline-size` for responsive layout
- Skeleton has `aria-hidden="true"` (decorative loading indicator)
- Avatar fallback shows initials when image unavailable
