# Form Selection

Checkbox, Switch, RadioGroup, and Select — accessible form selection controls.

## Checkbox

<ComponentPreview label="Checkbox states">
  <label style={{ display: "flex", alignItems: "center", gap: "var(--wui-spacing-2)", fontSize: "var(--wui-font-size-sm)", cursor: "pointer" }}>
    <input type="checkbox" defaultChecked={true} style={{ width: "16px", height: "16px", accentColor: "var(--wui-color-primary)", cursor: "pointer" }} />
    Checked
  </label>
  <label style={{ display: "flex", alignItems: "center", gap: "var(--wui-spacing-2)", fontSize: "var(--wui-font-size-sm)", cursor: "pointer" }}>
    <input type="checkbox" style={{ width: "16px", height: "16px", accentColor: "var(--wui-color-primary)", cursor: "pointer" }} />
    Unchecked
  </label>
  <label style={{ display: "flex", alignItems: "center", gap: "var(--wui-spacing-2)", fontSize: "var(--wui-font-size-sm)", cursor: "not-allowed", opacity: "0.5" }}>
    <input type="checkbox" disabled={true} style={{ width: "16px", height: "16px", accentColor: "var(--wui-color-primary)", cursor: "not-allowed" }} />
    Disabled
  </label>
</ComponentPreview>

```tsx
import { Checkbox } from "@weiui/react";

<Checkbox label="Accept terms and conditions" defaultChecked />
<Checkbox label="Remember me" checked={checked} onChange={(e) => setChecked(e.target.checked)} />
<Checkbox label="Disabled" disabled />
```

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| label | `string` | — | Visible label text |
| checked | `boolean` | — | Controlled checked state |
| defaultChecked | `boolean` | — | Initial uncontrolled checked state |
| disabled | `boolean` | `false` | Disables the checkbox |

### Accessibility

- Label is linked to the input via `htmlFor` / `id`
- Keyboard: Space to toggle
- Supports `aria-label`, `aria-describedby`


### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| label | `string` | — | Visible label text |
| checked | `boolean` | — | Controlled checked state |
| defaultChecked | `boolean` | — | Initial uncontrolled checked state |
| disabled | `boolean` | `false` | Disables the switch |

### Accessibility

- Uses `role="switch"` so screen readers announce it correctly
- `aria-checked` is set automatically from the checked state
- Keyboard: Space to toggle

</ComponentPreview>

```tsx
import { RadioGroup, RadioGroupItem } from "@weiui/react";

<RadioGroup defaultValue="a" aria-label="Fruit">
  <RadioGroupItem value="a" label="Apple" />
  <RadioGroupItem value="b" label="Banana" />
  <RadioGroupItem value="c" label="Cherry" />
</RadioGroup>
```

### RadioGroup Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| value | `string` | — | Controlled selected value |
| defaultValue | `string` | `""` | Initial uncontrolled value |
| onValueChange | `(value: string) => void` | — | Called when selection changes |

### Accessibility

- Container has `role="radiogroup"`
- Arrow keys navigate items; Space/Enter to select
- All items share the same `name` attribute

<ComponentPreview label="Select">
  <select style={{ minBlockSize: "44px", paddingInline: "var(--wui-spacing-3)", fontSize: "var(--wui-font-size-sm)", color: "var(--wui-color-foreground)", backgroundColor: "var(--wui-color-background)", border: "1px solid var(--wui-color-border)", borderRadius: "var(--wui-shape-radius-md)", cursor: "pointer", maxWidth: "240px", width: "100%" }}>

```tsx
import { Select, SelectTrigger, SelectValue, SelectContent, SelectItem } from "@weiui/react";

<Select value={lang} onValueChange={setLang}>
  <SelectTrigger aria-label="Language">
    <SelectValue placeholder="Pick a language" />
  </SelectTrigger>
  <SelectContent>
    <SelectItem value="en">English</SelectItem>
    <SelectItem value="fr">French</SelectItem>
    <SelectItem value="de">German</SelectItem>
  </SelectContent>
</Select>
```

### Select Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| value | `string` | — | Controlled selected value |
| defaultValue | `string` | — | Initial uncontrolled value |
| onValueChange | `(value: string) => void` | — | Called when selection changes |
| disabled | `boolean` | `false` | Disables the select |

### Accessibility

- Follows the ARIA Listbox pattern
- Keyboard: Space/Enter to open; Arrow keys to navigate; Escape to close
- `aria-expanded`, `aria-haspopup`, `aria-selected` managed automatically
