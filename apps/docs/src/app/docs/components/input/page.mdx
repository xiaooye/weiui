## Usage

```tsx
import { Input, Textarea } from "@weiui/react";

<Input placeholder="Email" />
<Input invalid placeholder="Invalid email" />
<Textarea placeholder="Message..." />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| invalid | `boolean` | `false` | Shows error state |
| disabled | `boolean` | `false` | Disables input |
| className | `string` | — | Additional classes |

All standard `<input>` HTML attributes are also supported.

## Accessibility

- Minimum 44px height touch target
- `aria-invalid` when invalid
- Focus ring on focus-within
- Supports native validation pseudo-classes
