# Date & Time

Calendar grid and date picker dropdown components.

## Calendar

Month view grid with day selection, navigation, and min/max constraints.

```tsx
import { Calendar } from "@weiui/react";

<Calendar onChange={(date) => console.log(date)} />

```

### Locale & first day of week

Month name and weekday headers come from `Intl.DateTimeFormat` ŌĆö pass a BCP-47 `locale` to localise the calendar. `firstDayOfWeek` (0 = Sunday, 1 = Monday, etc.) controls the column order.

```tsx
<Calendar locale="fr-FR" firstDayOfWeek={1} />
<Calendar locale="ja-JP" />
<Calendar locale="ar-EG" firstDayOfWeek={6} />
```

### Keyboard

The full month grid supports keyboard navigation:

- ArrowLeft / ArrowRight move one day
- ArrowUp / ArrowDown move one week
- Home / End jump to the first / last day of the current week
- PageUp / PageDown switch to the previous / next month
- Shift+PageUp / Shift+PageDown switch year
- Enter / Space selects the focused day
- Dates outside `minDate` / `maxDate` are skipped during navigation

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| value | `Date` | ŌĆö | Selected date |
| defaultValue | `Date` | ŌĆö | Initial date |
| onChange | `(date: Date) => void` | ŌĆö | Date selection handler |
| minDate | `Date` | ŌĆö | Earliest selectable date |
| maxDate | `Date` | ŌĆö | Latest selectable date |
| isDateDisabled | `(date: Date) => boolean` | ŌĆö | Predicate for disabling arbitrary dates (e.g. weekends, blackout days) |
| locale | `string` | `"en-US"` | BCP-47 locale tag for month / weekday names |
| firstDayOfWeek | `0 \| 1 \| 2 \| 3 \| 4 \| 5 \| 6` | `0` | Index of the first column (0 = Sunday) |
| disabled | `boolean` | `false` | Disables all interaction |

### Accessibility

- Calendar grid uses `role="grid"` with `aria-labelledby` pointing at the month heading.
- Each day button carries a full `aria-label` (e.g. `"Wednesday, April 15, 2026"`) so screen readers announce the date even when only the day number is visible.
- Previous/Next month navigation buttons have `aria-label="Previous month"` / `"Next month"`.
- Roving focus ŌĆö only the focused day is tabbable, matching the ARIA grid pattern.
- All day buttons and nav arrows meet the 44├Ś44px minimum touch target.
- Dates outside `minDate` / `maxDate` or rejected by `isDateDisabled` are skipped during keyboard navigation.

## DatePicker

Calendar inside a dropdown triggered by a button. The popover is anchored with Floating UI and rendered through a Portal, so it flips when close to the viewport edge and can't be clipped by scrolling ancestors.

<Preview label="DatePicker">
  <DatePickerDemo />
</Preview>

```tsx
import { DatePicker } from "@weiui/react";

<DatePicker placeholder="Pick a date" onChange={(date) => console.log(date)} />
<DatePicker value={date} onChange={setDate} minDate={new Date()} />

// Localised
<DatePicker locale="fr-FR" firstDayOfWeek={1} placeholder="Choisir une date" />
```

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| value | `Date` | ŌĆö | Selected date |
| onChange | `(date: Date) => void` | ŌĆö | Date selection handler |
| placeholder | `string` | `"Select date..."` | Trigger button text |
| minDate | `Date` | ŌĆö | Earliest selectable date |
| maxDate | `Date` | ŌĆö | Latest selectable date |
| isDateDisabled | `(date: Date) => boolean` | ŌĆö | Forwarded to the embedded Calendar for arbitrary date disabling |
| locale | `string` | `"en-US"` | BCP-47 locale tag forwarded to the Calendar |
| firstDayOfWeek | `0 \| 1 \| ŌĆ” \| 6` | `0` | First column of the week grid |
| disabled | `boolean` | `false` | Disables the picker |

### Accessibility

- Calendar uses `role="grid"` with `aria-labelledby` for month title
- Each day has full `aria-label` (e.g., "Wednesday, April 15, 2026")
- Navigation buttons: `aria-label="Previous month"` / `"Next month"`
- DatePicker trigger: `aria-haspopup="dialog"`, `aria-expanded`
- 44px minimum touch target on all day buttons and nav arrows
- State managed via `useReducer` for predictable month navigation
