# Layout Components

Container, Stack, Grid, Divider, Spacer, and AspectRatio ‚Äî six layout primitives for composing page structure.

---

## Container

Centers content horizontally with a max-width and horizontal padding.

```tsx
import { Container } from "@weiui/react";

<Container>
  <p>Centered content with max-width</p>
</Container>

// Override max-width
<Container maxWidth="600px">
  <p>Narrower container</p>
</Container>
```

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| maxWidth | `string` | `"1200px"` | CSS length for the inline max-size |
| children | `ReactNode` | ‚Äî | Content |

---

## Stack

A flex container that stacks children vertically (default) or horizontally.

```tsx
import { Stack } from "@weiui/react";

// Vertical stack (default)
<Stack gap={4}>
  <div>Item 1</div>
  <div>Item 2</div>
  <div>Item 3</div>
</Stack>

// Horizontal row
<Stack direction="row" gap={2}>
  <div>Left</div>
  <div>Right</div>
</Stack>

// Wrapping row
<Stack direction="row" wrap gap={2}>
  <div>Tag 1</div>
  <div>Tag 2</div>
  <div>Tag 3</div>
</Stack>
```

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| direction | `"row" \| "column"` | `"column"` | Flex direction |
| gap | `number` | `0` | Token-indexed spacing between children |
| align | `"start" \| "center" \| "end" \| "stretch"` | ‚Äî | `align-items` value |
| justify | `"start" \| "center" \| "end" \| "between"` | ‚Äî | `justify-content` value |
| wrap | `boolean` | `false` | Allow children to wrap to new rows |

---

## Grid

A CSS grid container with configurable columns and gap.

```tsx
import { Grid } from "@weiui/react";

// 3-column equal grid
<Grid columns={3} gap={4}>
  <div>Cell 1</div>
  <div>Cell 2</div>
  <div>Cell 3</div>
</Grid>

// Custom column template
<Grid columns="1fr 2fr" gap={4}>
  <div>Narrow</div>
  <div>Wide</div>
</Grid>
```

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| columns | `number \| string` | `1` | Number of equal columns, or a raw `grid-template-columns` string |
| rows | `number \| string` | ‚Äî | Number of equal rows, or a raw `grid-template-rows` string |
| gap | `number` | `0` | Token-indexed gap between cells |

---

## Divider

A horizontal or vertical separator line.

```tsx
import { Divider } from "@weiui/react";

// Horizontal (default)
<Divider />

// Vertical
<Stack direction="row">
  <div>Left</div>
  <Divider orientation="vertical" />
  <div>Right</div>
</Stack>
```

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| orientation | `"horizontal" \| "vertical"` | `"horizontal"` | Divider axis |
| decorative | `boolean` | `true` | When `true`, sets `role="none"`; when `false`, renders as a semantic `role="separator"` |

---

## Spacer

A flexible space element that fills available flex space.

```tsx
import { Stack, Spacer } from "@weiui/react";

<Stack direction="row">
  <div>Logo</div>
  <Spacer />
  <div>Actions</div>
</Stack>
```

### Props

Spacer takes no props ‚Äî it renders a `<div>` with `flex: 1 1 auto` to absorb leftover space.

---

## AspectRatio

Constrains children to a given aspect ratio.

```tsx
import { AspectRatio } from "@weiui/react";

// 16:9 (default)
<AspectRatio>
  <img src="/hero.jpg" alt="Hero" />
</AspectRatio>

// 4:3
<AspectRatio ratio={4 / 3}>
  <img src="/photo.jpg" alt="Photo" />
</AspectRatio>

// Square
<AspectRatio ratio={1}>
  <img src="/avatar.jpg" alt="Avatar" />
</AspectRatio>
```

### Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| ratio | `number` | `16 / 9` | Target aspect ratio as `width / height` |
| children | `ReactNode` | ‚Äî | Media or content to constrain |

---

## Accessibility

- All six primitives render semantic HTML and forward `...props`, so you can pass `role`, `aria-*`, and `data-*` attributes straight through.
- `Divider` with `decorative={false}` exposes `role="separator"` and supports `aria-orientation`; the decorative default hides it from assistive tech so layout dividers don't clutter announcements.
- Layout primitives use logical CSS properties (`padding-inline-*`, `margin-block-*`), so RTL documents mirror correctly with no extra configuration.
- `AspectRatio` uses the native `aspect-ratio` CSS property rather than a padding hack, so media inside stays sharp at any zoom level.
