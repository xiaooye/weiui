# Getting Started

## Installation

Install the token and CSS packages:

```bash
pnpm add @weiui/tokens @weiui/css
```

## Setup

Import the token CSS and base styles in your root layout:

```tsx
import "@weiui/tokens/tokens.css";
import "@weiui/css";
```

## First Component

```html
<button class="wui-button wui-button--solid">Click me</button>
```
