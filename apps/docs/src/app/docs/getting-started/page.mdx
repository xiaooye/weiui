# Getting Started

WeiUI is a three-layer design system. Start with the layer that matches how you work:

- **Just CSS?** Install `@weiui/tokens` and `@weiui/css`. Zero JavaScript.
- **React, unstyled?** Add `@weiui/headless` for accessible hooks and compound components.
- **React, styled?** Use `@weiui/react` — polished components on top of the two layers below.

## 1. Install

The quickest start uses the styled React layer. It depends on the other three packages, so installing it pulls them all in.

<PackageManagerTabs command="@weiui/react" />

## 2. Import tokens and base CSS

Add these two lines once, at the top of your app. Tokens define the color, spacing, and typography scales; the CSS layer ships reset + component styles.

```tsx
// app/layout.tsx (or equivalent)
import "@weiui/tokens/tokens.css";
import "@weiui/css";
```

Tokens default to system light/dark via `prefers-color-scheme`. To force a theme, add `class="dark"` or `data-theme="dark"` to `<html>`.

## 3. Render your first component

```tsx
import { Button } from "@weiui/react";

export default function Page() {
  return <Button>Hello WeiUI</Button>;
}
```

Prefer a CSS-only build? Skip the import and use classes directly:

```html
<button class="wui-button wui-button--solid">Hello WeiUI</button>
```
